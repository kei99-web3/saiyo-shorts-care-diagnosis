# 初回価格と決済手段の推奨

## 推奨初回価格

初回3件限定のモニター価格は 19,800円 を推奨。

理由:

- 9,800円だとB2B向け制作物として安すぎ、品質への信頼を落としやすい
- 29,800円以上だと実績ゼロの初回検証では心理ハードルが上がる
- 19,800円なら「試せる価格」と「ちゃんとした制作物」の両方を保ちやすい

## 通常価格への移行

- 初回3件: 19,800円
- 4件目以降: 39,800円
- LP/FAQ/計測シート込みの拡張版: 79,800円

## 決済手段

第一候補は Stripe Payment Links。
カード決済URLを個別に発行でき、YouTubeやメールからも案内しやすい。

第二候補は銀行振込。
介護施設や法人相手では、カード決済より請求書/振込の方が通りやすい場合がある。

運用は「Stripeリンク + 銀行振込も可」が最も無難。

最初の動画投稿時点では、LP上に価格は出さない。
無料診断後の返信で、相手の状況に合わせて初回モニター枠を案内する。
