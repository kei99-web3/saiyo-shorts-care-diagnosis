求人票URLをお送りいただき、ありがとうございます。
内容を拝見し、未経験者や転職検討者が応募前に気にしやすい点を中心に確認しました。

まず、応募前に残りやすい不安は次の3つです。

1. 初日に何をするのかが少し見えにくい
2. 誰がどのくらい教えてくれるのかが伝わりにくい
3. 失敗したときにどうフォローされるのかが分かりにくい

この不安を減らすShortsの切り口としては、次の3本が相性よさそうです。

1. 「介護スタッフの初日は何をする？」
2. 「未経験でも大丈夫な理由は、担当者がつくこと」
3. 「応募前に不安な方へ。見学だけでも確認できること」

求人票上では、「未経験歓迎」「丁寧に教えます」だけで止めずに、
初日の流れ、教育担当の有無、最初の2週間のフォローを1つの文章で見せると、応募前の不安が下がりやすくなります。

例:
「初日は先輩と一緒に、利用者さんの名前、1日の流れ、記録の見方を確認します。最初の2週間は担当者がつき、分からないことはその場で確認できます。」

もしこの方向で実際の採用Shortsまで作る場合は、
求人票をもとに9:16動画3本、字幕、投稿文、固定コメント、応募前FAQまでまとめて制作できます。

まずは初回モニター枠として、範囲を絞って低リスクに試せる形で案内できます。
必要でしたら、今回の求人票をもとに「3本の構成案」と「制作範囲」を具体化してお送りします。
