# 無料診断返信ワークフロー

## 基本方針

無料診断の返信は、求人票を見た相手が「この人はちゃんと見てくれている」と感じる品質を最低ラインにする。
返信の目的は、無料で価値を返しつつ、有料の採用Shorts 3本パックへ自然に進む理由を作ること。

## 返信の構成

1. 受付へのお礼
2. 求人票を見て気づいた応募前の不安3つ
3. その不安を減らすShorts切り口3本
4. 求人票上部の改善案
5. 有料制作に進む場合の内容
6. 次のアクション

## 初回5件ルール

初回5件は、Codexが返信文を作成したあと、送信前にユーザー確認を取る。
確認対象は「診断の見立て」「表現の強さ」「有料提案の自然さ」「価格提示の有無」。

6件目以降は、以下を満たす場合に確認なしで返信する。

- 求人票URLを確認できる
- 誇大表現がない
- 採用成功保証のような表現がない
- 相手の施設名、求人内容、募集職種に合わせた具体性がある
- 有料提案が押し売りに見えない

## 有料提案の出し方

無料診断の末尾で、次のように自然に案内する。

「もしこの方向で実際の採用Shortsまで作る場合は、求人票をもとに9:16動画3本、字幕、投稿文、固定コメント、応募前FAQまでまとめて制作できます。まずは初回モニター枠として、範囲を絞って低リスクに試せる形で案内します。」

## 返信品質チェック

送信前に次を確認する。

- 具体的な求人票の情報に触れている
- 「誰に」「何を」伝える動画かが明確
- 受け手が次に何をすればいいか分かる
- 値引き感ではなく、検証パッケージとして見える
- 文章が上から目線になっていない
